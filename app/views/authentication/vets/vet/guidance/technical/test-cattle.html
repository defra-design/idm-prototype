Guidance

Test for bovine viral diarrhoea (BVD) in cattle

Published 12 July 2022

Test type and sampling for beef cattle

To test for BVD in beef cattle, you must do a blood (serum) antibody test.

You should carry out 6 tests in up to 2 epidemiological groups. Where the primary or secondary epidemiological group number is less than 6, all animals should be sampled.

Read BVD Free guidance on how to test for BVD. if you need more information. If the farmer is a CHeCs member, you can either follow CHeCs guidance on how to test for BVD or follow BVD Free guidance.

Test type and sampling for dairy cattle

To test for BVD in dairy cattle, you must do a bulk milk test. You can choose with the farmer whether to do an antibody or PCR bulk milk test.

You can use the farm’s milk processor bulk milk test results. You cannot use the results of a test that was carried out before you applied for the review.

If you’re confident the farmer has the expertise to carry out bulk milk sampling and they agree, they can take the sample. The farmer would not do the test.

If you and the farmer want to do a blood serum antibody test instead of a bulk milk test, you can. The results would be accepted, but the £372 payment rate for a dairy cattle review will not change. The antibody test must be to the same standard as is specified for beef cattle.

Read BVD Free guidance on how to test for BVD. if you need more information. If the farmer is a CHeCs member, you can either follow CHeCs guidance on how to test for BVD or follow BVD Free guidance.

Test sample laboratory analysis

You must send the test sample for analysis to a laboratory that is one or both of the following:

  • ISO17025 accredited
  • UKAS accredited (UK accredited) for BVD testing

Find an approved laboratory to test for bovine viral diarrhoea (BVD) in cattle

Record and report test results

You must discuss the test results and any further action that needs to be taken with the farmer as part of the review.

You must:

  • record some test result information with Defra
  • include the test results as part of the written report you give to the farmer after the review, with appropriate recommendations

The test results can be used for other accreditation schemes that the farmer is a member of.
