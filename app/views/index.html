Customer identity will help users:

  • create a single set of sign-in credentials that can be used to access one or more Defra services.
  • by storing information about the user (and their business, if relevant) so that they do not have to re-enter it each time they want to use a new online service at Defra.
  • give other people (for example colleagues or agents) access to one or more of the online services they use.
