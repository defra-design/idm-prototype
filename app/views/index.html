This service will help users to

  • Create a single set of sign-in credentials that they can use to access one or more Defra services.
  • Store information about the user (and their business, if relevant) so that they do not have to re-enter it each time they want to use a new online service.
  • Give other people (for example colleagues or agents) access to one or more of the online services they use.
