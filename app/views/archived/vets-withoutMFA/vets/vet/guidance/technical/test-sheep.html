Guidance

Test for the effectiveness of worming treatments in sheep

Published 8 July 2022

Test type and sampling

You must use a worming treatment check to test how well a worming treatment has worked.

Follow detailed guidance on how to carry out this testing on the SCOPS website.

If the first round of sampling does not show worm eggs present, there is no need to carry out further sampling. You’ll still need to report the results of the initial egg count.

You must use a worming treatment that has one of the following active chemicals:

  • BZ
  • LV
  • ML
  • AD
  • SI

Test sample laboratory analysis

You must send the faeces samples for a composite worm egg count for analysis to a laboratory that conforms to any of the following:

  • ISO/IEC 17025 accredited
  • UKAS accredited (UK accredited) for a worm egg count
  • part of the APHA VetQAS worm eggs count proficiency scheme and has provided results defined by the z-score as satisfactory

Find an approved laboratory to test for the effectiveness of worming treatments in sheep

Record and report test results

You must discuss the test results and any further action that needs to be taken with the farmer as part of the review.

You must:

  • record some test result information with Defra
  • include the test results as part of the written report you give to the farmer after the review, with appropriate recommendations

The test results can be used for other accreditation schemes that the farmer is a member of.
